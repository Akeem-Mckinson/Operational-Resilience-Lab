# Operational Resilience Lab

## Overview

This project demonstrates the design, deployment, and management of a secure Azure infrastructure environment focused on operational resilience, security hardening, monitoring, and infrastructure automation.

## Technologies Used

- Microsoft Azure
- Terraform
- Linux Administration
- Azure Virtual Machines
- Network Security Groups (NSGs)
- Log Analytics
- Azure Monitor
- SSH Key Authentication
- GitHub

## Architecture

Components deployed:

- Resource Group
- Virtual Machine
- Network Security Groups
- Log Analytics Workspace
- Azure Monitoring
- SSH Access Controls
- Security Hardening Configuration

## Terraform Automation

Infrastructure deployment was validated and deployed using Terraform.

terraform init
terraform validate
terraform plan
terraform apply

## Skills Demonstrated

- Cloud Infrastructure Engineering
- Infrastructure as Code (IaC)
- Linux Administration
- Security Hardening
- Monitoring & Alerting
- Azure Resource Management
- Terraform Automation
