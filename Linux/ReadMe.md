# Linux Infrastructure Deployment

## Overview

This project demonstrates the deployment and validation of an Azure Linux virtual machine running Ubuntu 24.04 LTS.

## Objectives

- Deploy Azure Linux VM
- Connect via SSH
- Install Nginx
- Verify web server operation
- Validate Linux host configuration

## Evidence

- SSH login successful
- Nginx installation completed
- Web server validation
- Linux host verification
- Nginx service status active

## Skills Demonstrated

- Linux Administration
- SSH Management
- Nginx Deployment
- Azure Infrastructure
- System Validation
