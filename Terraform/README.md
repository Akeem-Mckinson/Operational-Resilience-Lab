# Terraform Infrastructure Deployment

## Overview

This project demonstrates Infrastructure as Code (IaC) using Terraform and Microsoft Azure.

The deployment automates the creation of Azure resources through Terraform configuration files and validates successful resource provisioning.

---

## Objectives

- Create Terraform project structure
- Initialize Terraform environment
- Validate Terraform configuration
- Generate execution plan
- Deploy Azure resources
- Verify resource creation in Azure Portal

---

## Technologies Used

- Terraform
- Microsoft Azure
- Azure Cloud Shell
- Azure Resource Groups
